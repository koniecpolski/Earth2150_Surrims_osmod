EarthC Documentation - EarthC Language Description

EarthC is a simple language heavily based on C. We use it to control game's objects, modify AI behaviors, create missions and many many more.

Basic structure of an EarthC script

BaseClassIdentifier "Script Name" {
   Consts
   Variable Definitions
   Functions
   States
   Events
   Commands
}

"Script Name" is a name which this script has in game
